## MVP Objective

Launch the APEX OS Short Engine MVP within 14 days.

Primary success criteria:

1. Automatically discover candidate viral signals.
2. Transform signals into DC 2100-compatible concepts.
3. Generate usable Scripts, Prompts, and Metadata.
4. Support a fast human video-generation loop.
5. Publish content and return analytics into the system.
6. Track YouTube Partner Program progress and growth velocity.

---

## MVP Principle

Automate intelligence before automating expensive generation.

The MVP automates:

- discovery
- analysis
- transformation
- Creative Pack generation
- publishing coordination
- analytics return

The MVP keeps video generation human-controlled.
